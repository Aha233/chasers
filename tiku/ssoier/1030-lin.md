# 1030-lin
## 题目  
**[题目描述]**  
对于半径为r的球，其体积的计算公式为V=4/3∗(πr^3)，这里取π=3.14。现给定r，即球半径，类型为double，求球的体积V，保留到小数点后2位。  

**[输入描述]**   
输入为一个不超过100的非负实数，即球半径，类型为double。  

**[输出描述]**  
输出一个实数，即球的体积，保留到小数点后2位。  

**[输入样例]**  
4  

**[输出样例]**  
267.95  

## 讲解  
[函数基础](C6.1函数基础)  

[数学函数]([1])  

## 代码  

```cpp
#include<stdio.h>
#include<math.h>
#define PI 3.14

int main()
{
	double r;
	scanf("%lf",&r);
	printf("%.2lf",4*PI*pow(r,3.0)/3);//调用pow函数
	return 0;
}
```
