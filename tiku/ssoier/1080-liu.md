# 1080
## 题目  
**[题目描述]**  
已知三个正整数`a，b，c`。现有一个大于1的整数x，将其作为除数分别除`a，b，c`，得到的余数相同。  

请问满足上述条件的x的最小值是多少？数据保证x有解。  

**[输入描述]**   
一行，三个不大于1000000的正整数`a，b，c`，两个整数之间用一个空格隔开。  

**[输出描述]**  
一个整数，即满足条件的x的最小值。  

**[输入样例]**  
300 262 205  

**[输出样例]**  
19  

## 讲解    
[循环结构](a)  
[条件语句](a)  

## 代码   

```cpp
#include<iostream>  

using namespace std;  

int main()  
{  
	int a,b,c;  
	cin>>a>>b>>c;  
	for(int i=2;;i++)  
		if(a%i==b%i&&b%i==c%i)  
		{  
			cout<<i<<endl;  
			break;  
		}  
	return 0;  
}
```
