# 1089
## 题目  
**[题目描述]**  
给定一个整数，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零，例如输入-380，反转后得到的新数为-83。  

**[输入描述]**   
输入共 1 行，一个整数N。
`-1,000,000,000 ≤ N≤ 1,000,000,000`。  

**[输出描述]**  
输出共 1 行，一个整数，表示反转后的新数。  

**[输入样例]**  
123  

**[输出样例]**  
321  

## 讲解    
[while循环](a)  
[条件语句](a)  

## 代码   

```cpp
#include<iostream>  

using namespace std;  

int main()  
{  
	int n;  
	cin>>n;  
	int flag=0;  
	if(n==0)  
	{  
		cout<<"0"<<endl;  
		return 0;  
	}  
	if(n<0)  
	{  
		cout<<"-";  
		n=-n;  
	}  
	while(n>0)  
	{  
		if(n%10==0&&flag==1)  
			cout<<"0";  
		else if(n%10>0)  
		{  
			flag=1;
			cout<<n%10;  
		}
		n/=10;  
	}  
	return 0;  
}  
```
