# 1010-lin
## 题目  
**[题目描述]**  
两个整数a和b分别作为分子和分母，既分数a/b，求它的浮点数值(双精度浮点数，保留小数点后9位)。  

**[输入描述]**   
输入仅一行，包括两个整数a和b。  

**[输出描述]**  
输出也仅一行，分数 a/b 的浮点数值（双精度浮点数，保留小数点后9位）。  

**[输入样例]**  
5 7  

**[输出样例]**  
0.714285714  

## 讲解  
c++基本输入输出格式  

[类型转换](C2.1基本内置类型&&C4.11类型转换)  

[c++ 算数运算符](C4.2算术运算符)  


## 代码  

```cpp
#include<stdio.h>
#include<iostream>

using namespace std;

int main()
{
	int a,b;
	scanf("%d %d",&a,&b);
	printf("%.9f",(double)a/b);
	return 0;
}
```
