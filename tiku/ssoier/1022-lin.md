# 1022-lin
## 题目  
**[题目描述]**  
将一个整型变量的值赋给一个布尔型变量，再将这个布尔型变量的值赋给一个整型变量，得到的值是多少？  

**[输入描述]**   
一个整型范围内的整数，即初始时整型变量的值。  

**[输出描述]**  
一个整数，经过上述过程后得到的结果。  

**[输入样例]**  
3  

**[输出样例]**  
1  

## 讲解  
[类型转换](C2.1基本内置类型&&C4.11类型转换)  

## 代码  

```cpp
#include<stdio.h>

int main()
{
	int i;
	bool j;
	scanf("%d",&i);
	j=i;
	i=j;
	printf("%d",i);
	return 0;
}
```
