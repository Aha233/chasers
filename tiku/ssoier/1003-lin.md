# 1003-lin  
## 题目  
**[题目描述]**  
读入三个整数，按每个整数占8个字符的宽度，右对齐输出它们，按照格式要求依次输出三个整数，之间以一个空格分开。  

**[输入描述]**   
只有一行，包含三个整数，整数之间以一个空格分开。  

**[输出描述]**  
只有一行，按照格式要求依次输出三个整数，之间以一个空格分开。  

**[输入样例]**  
123456789 0 -1  

**[输出样例]**  
123456789       0      -1  

## 讲解  
c++基本输入输出  

## 代码  

```cpp
#include<stdio.h>
#include<iostream>

using namespace std;

int main()
{
	int a=0,b=0,c=0;
	scanf("%d %d %d",&a,&b,&c);
	printf("%8d %8d %8d",a,b,c);
	return 0;
}
```
